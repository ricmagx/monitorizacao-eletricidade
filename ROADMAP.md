# Roadmap

## Passo 1

Definir o contrato atual:

- fornecedor
- potencia contratada
- tarifa simples ou bihorario
- custo real recente

## Passo 2

Obter o primeiro conjunto de dados mensais:

- exportacao E-REDES
- totais mensais por periodo

## Passo 3

Criar o catalogo inicial de comercializadores:

- `3 a 5` opcoes
- fonte guardada
- data de recolha guardada

## Passo 4

Implementar calculadora de custo:

- custo do contrato atual
- custo das alternativas
- ranking

## Passo 5

Publicar resultado no Home Assistant:

- melhor fornecedor atual
- poupanca potencial
- data da ultima simulacao
