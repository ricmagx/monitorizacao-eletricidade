  <name>Task 1: Funções de análise multi-ano em data_loader.py</name>
  <files>src/web/services/data_loader.py, tests/test_web_data_loader.py</files>
  <behavior>
    - build_consumo_multi_ano(consumo_data: list) -> dict:
        Input: lista de {"year_month": "YYYY-MM", "vazio_kwh": float, "fora_vazio_kwh": float}
        Output: {"anos": ["2023","2024","2025"], "meses": ["Jan","Fev",...,"Dez"],
                 "datasets": [{"ano":"2023","vazio":[...],"fora_vazio":[...]}, ...]}
        Cada dataset tem 12 entries (None para meses sem dados).
        Anos ordenados ASC; meses sempre 01-12 como índice.
    - build_resumo_anual(consumo_data: list, comparacoes_history: list | None) -> list:
        Output: [{"ano":"2023","consumo_total_kwh":float,"custo_total_eur":float|None}, ...]
        custo_total_eur é soma de current_supplier_result.total_eur de comparacoes_history
        para esse ano (None se sem dados de comparação).
    - build_comparacao_meses(consumo_data: list, comparacoes_history: list | None,
                             ano1: str, ano2: str, mes: str) -> dict:
        Output: {"mes":"03","ano1":"2023","ano2":"2024",
                 "consumo_ano1":float|None,"consumo_ano2":float|None,
                 "custo_ano1":float|None,"custo_ano2":float|None}
        mes no formato "MM" (zero-padded). Retorna None nos campos se dados inexistentes.
    - Testes: valores corretos com 3 anos de dados, meses em falta (None), ano sem comparacoes (custo_total_eur=None)
  </behavior>
  <action>
    Adicionar as 3 funções ao final de src/web/services/data_loader.py.

    build_consumo_multi_ano: agrupa consumo_data por ano extraindo year[:4]; para cada ano
    cria lista de 12 valores (índice 0=Jan..11=Dez) inicializada a None; preenche vazio+fora_vazio.

    build_resumo_anual: agrupa consumo_data por ano para consumo_total_kwh; se comparacoes_history
    fornecida, agrupa por year_month[:4] e soma total_eur de current_supplier_result; join por ano.

    build_comparacao_meses: filtra consumo_data por "YYYY-MM" == f"{ano}-{mes}" para cada ano;
    faz o mesmo em comparacoes_history para custo. Retorna dict com 6 campos.
